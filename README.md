## Installation
Add this on composer repositories section:
```json
{
    "type": "vcs",
    "url": "https://gitlab.pdmfc.com/pdmfc/short-url-package.git"
}
```


Require this package with composer:

```shell
composer require pdmfc/shorturl:*
```

# Configuration

```shell
php artisan vendor:publish --tag="short_config"
```

# Usage
```code
$shortUrlClient = new ShortUrlClient;
```
# Methods

```json
{
    "message": "Access is not allowed"
}
```

```code
$shortUrl = $shortUrlClient->createUrl($params);

domainUrl – Short url domain (optional) (ex: s.pdm.pt)
originalUrl – Url where you will be redirected (Mandatory)
liveTime – Short Url live tima (optional) (if not declared will be zero and the url never expires)
active – Define if the link is active or not (optional) (active by default)
shortUrl – User can be define a personalized short url code (ex: campanha_natal) (opcional). If not filled in, the system will create one according to the defined parameters
```
